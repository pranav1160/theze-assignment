## 🚀 Description

**Ball Flip** is a fast-paced iOS arcade game built using **SwiftUI** and **SpriteKit**.

* Control a ball that flips between **top** and **bottom**
* Avoid incoming obstacles
* Score increases over time
* Game ends instantly on collision

## ⚡ Core Mechanic

* Tap to **instantly switch position**
* Pure reflex-based gameplay

## 🛠 Tech Stack

* SwiftUI
* SpriteKit
